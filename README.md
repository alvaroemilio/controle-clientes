# controle-clientes

- O Controle de Clientes é uma aplicação voltada para o gerenciamento de clientes e acompanhamento de valores pendentes, permitindo maior controle sobre operações de crediário e contas a receber.
  O sistema foi desenvolvido com foco em simplicidade, organização dos dados e facilidade de manutenção.

Sistema web desenvolvido com React + Vite no frontend e Node.js + Express no backend, utilizando MySQL como banco de dados.
Frontend -> React, React DOM, Vite, Axios, Bootstrap, React Router DOM, ESLint
Backend -> Node.js, Express, CORS, MySQL2

- Funcionalidades

Cadastro de clientes
Consulta de clientes cadastrados
Controle de contas a receber
Registro de pagamentos
Histórico financeiro
Envio de informações
Atualização de dados cadastrais
